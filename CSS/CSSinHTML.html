CSS in HTML

  1. Internal: Su dung the style trong file HTML
  2. External: Viết một file css riêng rồi link vào HTML
  3. Inline: Viết attribute thẳng vào dòng lệnh

Mức độ ưu tiên trong CSS

  1. Internal và External: gọi cái nào sau thì áp dụng cái đó trước.
  2. Ưu tiên: Inline 1000 điểm > #id 100 điểm > .class 10 điểm > tag 1 điểm

1. Absolute units?

2. Relative units?

  • %

  • vh: chiều dọc của trình duyệt (theo đơn vị %)
  • vmin
  • vmax
  • ex
  • ch
